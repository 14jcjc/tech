## katsu について

長年、データサイエンス／データ分析を軸に、エンジニアとして活動してきました。  
分析や推論そのものを楽しむことが、昔からの関心事です。

現在はフリーで、データサイエンスに関する情報発信を行いながら、いくつかの個人プロジェクトに取り組んでいます。

上智大学および同大学院 (数学専攻) 修了。  
データサイエンス分野を中心に、システムエンジニア／データサイエンティストとして20年以上の実務経験があります。  
懸賞金付きデータサイエンス・コンテストにて2度の準優勝を獲得しました。

R を使ったデータ分析やプログラミングが特に好きで、当サイトでも R や SQL に関する話題を中心に扱っています。

## 当サイトについて

当サイトでは、R と SQL を中心に、データサイエンスの実践に役立つ情報を発信しています。  
特に、演習問題を通じて、**実践的なスキルを効率よく習得できる**よう工夫しています。  

演習問題に取り組む際は、以下の順序で記事をご覧いただくと、スムーズに進められます。

1. **[書籍レビュー]({{% ref "/ds-drills/data-tf/base/book-review" %}} "書籍レビュー")**

   必要な知識や準備について解説
